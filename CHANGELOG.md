# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- 全局快捷键翻译功能（默认：Cmd + Shift + T）
- 自动获取选中文本（使用辅助功能 API）
- OpenAI compatible API 翻译支持
- 自定义 API 端点配置
- 悬浮窗口显示翻译结果
- 设置界面（API Key、目标语言、API 端点配置）
- 菜单栏集成
- Keychain 安全存储 API Key
- 加载状态和错误提示
- 单元测试覆盖核心功能

### Technical
- Swift Package Manager 项目结构
- SwiftUI + AppKit 混合架构
- Carbon 框架全局快捷键支持
- ApplicationServices 辅助功能 API 集成
